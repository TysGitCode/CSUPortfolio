
# Edge URL Selector ‚Äî Guest Profile Launcher (Qt / PySide6)

A modern, professional desktop utility that lets you **select multiple URLs**, **drag to set launch order**, and open them in **separate Microsoft Edge windows** using **guest-like profiles** (`--user-data-dir`) ‚Äî ideal for quickly launching environment-specific portals (PRD / TST / TRN / PP1) without session collisions.

This was created as a solution for an issue on the infor team that prevented us from having more than 2 enviornments open at a time. 

> Built with **PySide6 (Qt)** for a polished UI and scalable model/view architecture.

---

## ‚ú® Highlights

- ‚úÖ **Modern UI** (Qt / PySide6) with optional **Material theme**
- ‚úÖ **Checkbox selection** (click row or checkbox; Spacebar toggles)
- ‚úÖ **Drag-and-drop launch ordering** (launches exactly in the order you arrange)
- ‚úÖ **Fast search/filter** (host, env, path, full URL)
- ‚úÖ **ENV ‚Äúchips‚Äù** (PRD/TST/TRN/PP1 visual cues)
- ‚úÖ **Launch delay control** (slider to pace Edge window launches)
- ‚úÖ **Copy actions + context menus**
- ‚úÖ **Config persistence** (Edge path, selections, delay, drag order)
